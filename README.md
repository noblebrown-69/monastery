# 🗡️ Monastery

**A lightweight, portable, old-school C++ word processor inspired by Microsoft Word 6.0 (1994).**

Built for Linux (Ubuntu/Mint) with a warm sepia parchment aesthetic. Single executable. Zero bloat. Lives in Dropbox and runs anywhere.

![Monastery Screenshot](https://github.com/noblebrown-69/monastery/raw/main/screenshot1.1.png)

## Features

- Classic Word 6.0-style toolbar (B/I/U, alignment, bulleted & numbered lists)
- Rich-text editing with HTML save/load (compatible with Word, LibreOffice, etc.)
- Auto-save every 30 seconds to `Docs/Monastery_AutoSave.html`
- Live word count in status bar
- Print support
- Simple page-break insertion
- Basic spell-check highlighter (red underline)
- Warm sepia parchment + leather theme (matches Aureus)
- Modular `MonasteryEditor` widget — ready to drop into Aureus later
- Single-file AppImage or tiny executable — truly portable
